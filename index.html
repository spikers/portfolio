Back End Web Developer

GitHub

I learned HTML and CSS a decade ago. I became proficient at front end technologies and began building websites in plain HTML and CSS using Windows Notepad.

Despite my web experience, I attended California State Polytechnic University, Pomona as a Civil Engineering major. After years of tweaking pages as hobbies, I was elected to become my Earthquake Club's webmaster, where I was introduced to Wordpress. I created several custom front-end elements for my club and maintained the website for a year.

At this time, I joined a startup, SocialTagg, LLC, as a Quality Assurance Tester. SocialTagg was a social event platform, much like Eventbrite, focused on connections at events. During this time, I submitted several hundreds of bugs using Assembla, meanwhile, I created custom HTML and CSS pages for our promotional landing pages. The back end of this application was node.js and express.js

Since then, I've competed in hackathons, like TechCrunch's Disrupt, CES, Money20/20, AT&T, and AngelHack. I've built applications in multiple languages, including Android, C#, and Javascript.
