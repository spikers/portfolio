I'm a software developer with specialization in Windows, particularly C#, ASP.NET, MVC, and T-SQL. In the back end, I've built all types of sites, including full fledged websites, I've created customizations for existing websites, and I've created utilities for management.

Among sites I've built from scratch, includes Siebert's Blog, shown below, and an document upload portal for Credit Track. The blog was built using WordPress while the main site (which was not have the same back end as the blog) was built with Sitefinity. The Document Upload Portal was fully built using C#.NET MVC, which used token authentication for login. The documents were directly integrated into our proprietary software.

I've also integrated several features into existing or legacy codebases, including most of my work with CreditTrack, which was proprietary software for commercial loan origination and Orange County Transportation Agency, which used Ektron, a uncommon CMS system.

In the front end, I'm well-versed in HTML, CSS, and Javascript, React, as well as ASP Web Forms. I've created API requests and responses and I've created all types of functionality across front end, back end, and database. For an example of front end, see Ipsum Galore's codebase.

As far as the database, most of my work at CreditTrack was in the database. I've wrote queries using T-SQL daily and I've used Entity Framework heavily in all kinds of situations.
